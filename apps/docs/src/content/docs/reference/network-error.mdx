## Symptom

After email/password sign-in, the server succeeds but the browser shows:

- `NetworkError when attempting to fetch resource`

## Why it happens

Some sign-in endpoints respond with redirects. If a client `fetch()` follows the redirect automatically, you can end up trying to parse a non-JSON response.

- Uses manual redirect for the sign-in/sign-up POST endpoints
- Treats redirect responses as success
